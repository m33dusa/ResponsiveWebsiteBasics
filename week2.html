WEEK 2 - CSS

  1. CSS

    2. CSS - stands for Cascading Style Sheets, it is a derived language that controls how webpages look on the browser. HTML, CSS and Javascript make up web technologies.

  2. Anatomy of a CSS

    3. Selectors - Selectors are patterns used to target HTML elements you want to style/modify.

    Property - Properties along with some value input create a delcration.

    Value - The value is simply, the desired trait you want your HTML element to embody.

  3. Pseudo-Class Selectors & Other Selectors

    4. Pseudo class selectors are CSS selectors with a colon preceding them. You can learn more about Pseudo Class Selectors on CSS-Tricks. In addition to the pseudo-class selectors, there are numerous other types of selectors, including parent ("<"), child (">") and descendant (blank space) selectors and a universal selector(*{}).

    Selectors can be combined, nested, merged and they can inherit the properties of other selectors. There is a preference hierarchy to selectors, your HTML element will take on the value of the most specific css indicator. The CSS preference hierarchy goes as follows: Style, ID, Class/Pseudo-Class and # of Elements.

  4. The Box Model

    5. By default, the web browser wraps every HTML element in a rectangular box, the CSS box model. The box has the following customizable CSS properties: height, width, border, margins, padding and content. According to the box model margins are cumulative. The universal selector is used to standardize settings across different viewports.

  5. Positioning

    6. The position property determines the position of content on the page and utilizes the following values:

    • Fixed - The fixed position removes content from the normal flow of the page and are not affected by scrolling.
    • Static - The default browser position places content accoding to the normal flow of the document.
    • Relative - The relative position describes the new placement of content relative to its normal/default position.
    • Absolute - The absolute position does not exist in the normal document flow. It automatically sets at the top-left corner of the page.
    • Sticky - The sticky position combines elements of the relative and fixed position.
    • Float - In addition to the regular positioning methods, the float method gravitates content to the "right", "left", "center", "top" or "bottom" of the page
    • Z-Index - The z-index is another determines the layer of a content. Higher z-index items are placed above lower indexed items.

  6. Media Queries

    7. The @media querie is used to target styles at specific media, usually, screen or print. The @media helps optimize content layout on different screen sizes, sizes ranging from mobile screens to large desktop screens.

  7. Responsive Design

    8. Responsive website design uses fluid, proportion-based grids, flexible images and CSS media queries to adapt a site's layout to the viewing environment. Responsive layouts use the 12-column grid model and uses % to determine the width of each grid.

  8. Twitter Bootstrap

    9. Twitter Bootstrap or simply Bootstrap is an open-source HTML, CSS and JavaScript Framework that is used in front-end web development to easily create mobile-first, responsive websites quickly.
