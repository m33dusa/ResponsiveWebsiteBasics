  1. Development Environment

    2. Atom - Atom is a free and open source text and source code editor for Windows, Linux and OS X. Atom is written in Javascript.

    Sublime Text - Sublime Text is a sophisticated text editor for code that operates on Windows, Linux and OS X. A version of Sublime Text can be downloaded for free, however, you will need to purchase the licensed version to continue using Sublime Text. Sublime Text is written in C++ and Python.

    Notepad++ - Notepad++ is a text editor and source code editor for Windows. Notepad++ is written in C++.

    TextWrangler - TextWrangler is a text and source code editor for OS X. TextWrangler is available for free download. TextWrangler is written in C.

  2. HTML

    3. HTML stands for Hypertext Markup Language. HTML is not a programming language like Python, Java, C++, C#, R, etc.

    • HTML is used to display documents on the world wide web ("web").

    • There are several syntax rules developed by the World Wide Consortium ("W3C"), for writing HTML files. In addition to these syntax rules, a HTML file has to be saved with the ".html" or ".htm" extension.

    • TAGS: The angle brackets < > mark the beginning and end of a tag. Week 1 covered the navigation tag, <nav>, image tag <img> and ordered <ol> and unordered <ul> list tags. Check out this list of html tags.

    • STRUCTURE: HTML uses the <html>, <style>, <title>, and <body> tags to properly structure content on your page.
